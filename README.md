# Airline Passenger Satisfaction Analysis

## Project Overview

This project analyzes airline passenger satisfaction data using Python, SQL, and Tableau. The goal is to identify the factors influencing customer satisfaction and generate business insights that could support operational improvements.

---

## Business Questions

- How do flight delays affect passenger satisfaction?
- Which travel class has the highest satisfaction rate?
- Does seat comfort impact customer experience?
- Which factors contribute most to customer satisfaction?
